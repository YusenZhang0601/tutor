# Security Policy

## Supported Versions

The public `main` branch is the supported version of this template.

## Reporting A Vulnerability

Please report security issues by email:

zhangyswx@163.com

Include a concise description, reproduction steps if available, and the affected files or commands.

## Privacy Expectations

This template is intended to stay free of private learner data. If you find personal notes, credentials, local machine paths, or private research artifacts in the public repository, please report them as a security/privacy issue.
