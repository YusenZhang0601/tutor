# onboard — 建立学习者画像

目标：通过对话访谈，填好 `.tutor/config/learner-profile.md`，让后续所有会话能针对个人。

## 步骤
1. 先读全局记忆与现有 profile，避免重复问已知信息（如学习者目标、背景、节奏和已有项目上下文）。
2. 逐项访谈（一次问 1–2 个，别一次性轰炸）：
   - 想学什么学科/项目？为什么、用来做什么？
   - 每个目标的**目标深度**：入门理解 / 能动手 / 能改真实代码 / 精通？
   - 现有基础、已知薄弱处、过去在哪卡住？
   - 可投入节奏（每次多长、每周几次）？
   - 讲法偏好（先直觉还是先形式化、多例还是多推导）？
3. 把答案写入 `learner-profile.md` 对应小节。
4. 确认无误后，建议下一步 `/plan <学科>`。

## 原则
- 访谈本身也是建立信任的过程，语气放松、像伙伴。
- 不臆测：信息缺了就问，别填默认值。
